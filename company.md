# Acme Corp - How We Work

> This file gives Claude context about Acme Corp so it can help team members more effectively. It covers who we are, what we build, how we work, and key guardrails.

## Who We Are

Acme Corp is a SaaS company building project management tools for distributed teams. Founded in 2020, we serve 5,000+ customers across 40 countries.

**Mission:** Make remote collaboration feel effortless.

**Product:** Acme Workspace - a real-time project management platform with task boards, docs, and async video.

## Values

1. **Ship fast, learn faster** - Bias toward action. Perfect is the enemy of good.
2. **Default to async** - Write it down. Respect time zones.
3. **Customers first** - Every decision starts with "how does this help customers?"
4. **Own your outcomes** - No finger-pointing. If you see a problem, fix it.
