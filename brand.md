# Acme Corp Brand Guide

> This file gives Claude the full Acme Corp brand system so it can generate on-brand content, UI, and design decisions.

## Colors

### Primary Palette

| Name | Hex | Usage |
|------|-----|-------|
| Slate | #1e293b | Headers, nav, dark backgrounds |
| Blue | #3b82f6 | Primary CTAs, links, active states |
| Emerald | #10b981 | Success, secondary actions |
| Amber | #f59e0b | Warnings, accent, highlights |
| Cloud | #f8fafc | Page backgrounds, cards |
| White | #ffffff | Card backgrounds, text on dark |

### Semantic Colors

| Token | Hex | Usage |
|-------|-----|-------|
| --error | #ef4444 | Error states, destructive actions |
| --success | #10b981 | Success messages, positive indicators |
| --warning | #f59e0b | Warnings, attention needed |
| --info | #3b82f6 | Informational callouts |

## Voice & Tone

- Clear and direct. No jargon.
- Friendly but not bubbly.
- Confident, not salesy.
- When in doubt, be helpful.
